# 🥷 נינג'ה קיטשן - PWA

## הוראות העלאה ל-Vercel (5 דקות)

### שלב 1 - GitHub
1. כנס ל-github.com → "New repository"
2. קרא לו `ninja-kitchen`
3. העלה את כל הקבצים מהתיקייה הזו

### שלב 2 - Vercel
1. כנס ל-vercel.com → "Sign up" (חינם)
2. לחץ "Add New Project"
3. חבר את ה-GitHub שלך ובחר את `ninja-kitchen`
4. לחץ "Deploy" — זהו!

### שלב 3 - התקנה בטלפון
**אנדרואיד (Chrome):**
- פתח את האתר ב-Chrome
- לחץ על 3 הנקודות למעלה
- "הוסף למסך הבית"

**אייפון (Safari):**
- פתח את האתר ב-Safari
- לחץ על כפתור השיתוף (ריבוע עם חץ)
- "הוסף למסך הבית"

## הרצה מקומית (אופציונלי)
```
npm install
npm run dev
```
